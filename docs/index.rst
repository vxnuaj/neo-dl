Nue
===============================
This is Nue! A deep learning framework designed for simple implementation of models, geared towards entry level learners.

Built with obsession by `vxnuaj <https://twitter.com/vxnuaj>`_

.. important:: This package is still under development

      I'm building this framework as a means to continue to get deeper into deep learning and maximize my learnings.
      
      Bear with me as a figure things out!

Quickstart
*************

1. PIP it!
::
   $ pip install nue

2. Test out any of the following models!

   - :ref:`Linear Regression <models.linreg>`
   - :ref:`Logistic Regression <models.logreg>`
   - :ref:`A Feedforward Neural Network for Classification<models.nn>`
